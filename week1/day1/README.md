# 💡 Fundamental SQL Queries

This section focuses on learning how SQL queries are used to work with data in relational databases. It covers how to retrieve, filter, organize, and analyze data using different SQL concepts and query techniques.

---

## 🧠 Key Concepts Practiced

### 🔹 Data Retrieval

* Using SELECT to access data
* Choosing required columns
* Applying conditions using WHERE

### 🔹 Pattern-Based Search

* Using LIKE for searching data
* Working with wildcards (`%`, `_`)

### 🔹 Working with Dates

* Applying conditions on date fields
* Extracting year and month

### 🔹 Data Aggregation

* Using COUNT, SUM, AVG, MIN, MAX
* Summarizing data for analysis

### 🔹 Grouping Data

* Using GROUP BY to organize records
* Applying aggregate functions on grouped data

### 🔹 Filtering Groups

* Using HAVING to filter grouped results

### 🔹 Sorting Results

* Using ORDER BY
* Sorting data in ascending and descending order

### 🔹 Combining Data

* Using JOINs to combine tables
* Handling unmatched data with LEFT JOIN

### 🔹 Advanced Querying

* Writing subqueries
* Using correlated queries for complex conditions
