# SQL JOIN Queries Practice

This section focuses on strengthening SQL JOIN concepts using relational database tables. It covers how multiple tables are connected and how data can be retrieved, compared, and analyzed using different types of JOIN operations.

---

## 🧠 Key Concepts Practiced

### 🔹 INNER JOIN
- Combining matching records from multiple tables
- Retrieving related student, course, and instructor data

### 🔹 LEFT JOIN
- Displaying all records from the left table
- Handling unmatched rows using NULL values

### 🔹 RIGHT JOIN
- Displaying all records from the right table
- Understanding reverse join behavior

### 🔹 FULL OUTER JOIN
- Retrieving matched and unmatched rows from both tables
- Identifying missing relationships between datasets

### 🔹 CROSS JOIN
- Generating all possible combinations between tables
- Understanding Cartesian products

### 🔹 Working with NULL Values
- Detecting missing instructor assignments
- Finding students without enrollments
- Identifying courses without registrations

### 🔹 Multi-Table JOIN Operations
- Combining students, courses, enrollments, and instructors
- Building complete relational reports
